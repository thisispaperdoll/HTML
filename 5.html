자주 쓰는 코드

  • p 태그 : p는 paragraph의 약자로 '문단'을 의미하고 p, /p 태그를 두 번 사용하면 줄바꿈 효과를 낼 수 있다. 하지만 문단이므로 사이 간격이 벌어진다.
  • br 태그 : br은 break의 약자고 열고 닫는 태그가 존재하지 않아 br만 쓰면 된다.
  • strong 태그 : 특정 텍스트에 주의를 끌기 위해서 글씨가 굵게 표시가 된다.
    • 코딩은 정말 중요하다고!
  • em 태그 : 주위 텍스트와 비교했을 때 해당 부분을 기울임꼴로 강조 표시가 된다.
    • 코딩은 정말 중요하다고!
  • a href 태그 : 특정 URL로 이동하고 싶을 때 누르면 사용할 수 있다.
    • 네이버로 들어가자
  • ul 태그 : unordered list의 약자로 지금처럼 나열하면서 설명할 때 많이 사용한다.
    • 예1
    • 예2
  • table border 태그 : 표를 만들 수 있는 태그이다.
  • tr 태그, th 태그, td 태그 : tr은 table row의 약자, th은 table head, td는 table data의 약자로 table border태그 안에서 같이 사용될 수 있다.
